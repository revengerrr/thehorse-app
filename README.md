## 🎯 **What is TheHorse.app?**

TheHorse.app is a **doom scroll blocker generator** that creates custom userscripts to help you stop wasting time on social media.

After your set time limit, a fullscreen meme image appears with sound effects to remind you to **touch grass** 🌿

> 💡 **Inspired by** [@ravarcheon's viral horse script](https://x.com/ravarcheon/status/2028109714706575623) that blocks your Twitter feed with a horse image.

## 🚀 **How It Works**

```
1. Pick your blocker image 🖼️
2. Set your time limit ⏱️
3. Select platforms to block 🌐
4. Generate & copy script 📋
5. Paste into Tampermonkey 🔧
6. Doom scroll → Get blocked! 🐴
```

### **Timeline Example (2 min limit)**

| Time | What Happens |
|------|--------------|
| 0:00 - 1:00 | Scroll normally |
| 1:00 - 2:00 | Blocker fades in (0% → 100%) |
| 2:00+ | Fully blocked + sound plays 🔊 |

## 📥 **Installation**

### **Step 1: Install Userscript Manager**

| Browser | Recommended |
|---------|-------------|
| Chrome | [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) |
| Firefox | [Tampermonkey](https://addons.mozilla.org/firefox/addon/tampermonkey/) |
| Edge | [Tampermonkey](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd) |
| Safari | [Userscripts](https://apps.apple.com/app/userscripts/id1463298887) |

### **Step 2: Generate Your Script**

1. Go to [**thehorse.app**](https://thehorse.app)
2. Pick your blocker, time, and platforms
3. Click **Generate My Script**
4. Click **Copy to Clipboard**

### **Step 3: Install Script**

1. Open Tampermonkey → **Create new script**
2. Delete everything
3. Paste your script
4. **Ctrl+S** to save
5. Done! 🎉
